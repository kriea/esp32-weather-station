# Embedded Weather Display

A smart weather station built on ESP32-C3 SuperMini with real-time updates, NTP time sync, and LVGL GUI.

## Features
- NTP-synchronized clock
- WiFi connectivity
- ST7789 1.54" display (240Ã—240)
- LVGL

## Hardware
- ESP32-C3 SuperMini
- ST7789 SPI display (1.54")
- WiFi connectivity (integrated)

## Setup
1. Clone repo
2. Configure WiFi credentials in `config.example.h` and rename it `config.h`
4. Upload to ESP32
